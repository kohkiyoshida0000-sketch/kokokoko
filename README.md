# kokokoko

Ink Link の静的UIデモです（タトゥー予約 + 運用機能 + コミュニティ機能）。

## 実装済み機能（デモ）
- オープニング演出（`Welcome to InkLink` タイピング後に入場）
- アーティストプロフィール登録（料金/駅名公開/SNS/専門スタイル）
- 予約作成（DMハンドル・デザイン状況共有）
- 3日前の下絵確認リマインド、前日顧客リスト
- 金額/地域検索、現在地取得ボタン
- ブラックリスト管理
- フラッシュ投稿販売（手数料5%表示）
- 顧客/アーティストのカレンダー共有メモ
- Stripe決済UI（月額4,980円 / 初月無料トライアル表示、公開鍵未設定時はデモ決済保存）
- クーポンコード入力（`GARAGE1717` または `GARAGE１７１７` で適用）
- 彫り師↔顧客DMログ（ローカル保存）
- フォロー機能、テキスト投稿、作品投稿、メンション表示

## 使い方
1. `index.html` をブラウザで開く
2. オープニング表示後「ホームページへ入る」を押す
3. 各フォームを入力して送信

## Stripe連携メモ
- 実運用にはサーバー側でCheckout Session生成を行ってください。
- このデモでは `window.INKLINK_STRIPE_PUBLIC_KEY` が未設定の場合、決済を「デモ完了」で保存します。
