# kokokoko

Ink Link の静的UIデモです（タトゥー予約 + 運用機能）。

## 実装済み機能（デモ）
- アーティストプロフィール登録（料金/駅名公開/SNS/専門スタイル）
- 大分類→詳細の専門スタイル選択
- 予約作成（DMハンドル・デザイン状況共有）
- 3日前の下絵確認リマインド
- 前日顧客リスト表示
- 金額/地域（駅名）検索、現在地取得ボタン
- ブラックリスト管理
- フラッシュ投稿販売（手数料5%表示）
- 顧客/アーティストのカレンダー共有メモ
- Stripe決済UI（公開鍵未設定時はデモ決済保存）
- 彫り師↔顧客DMログ（ローカル保存）

## 使い方
1. `index.html` をブラウザで開く
2. 各フォームを入力して送信
3. 右下ロゴボタンでトップへ戻る

## Stripe連携メモ
- 実運用にはサーバー側でCheckout Session生成を行ってください。
- このデモでは `window.INKLINK_STRIPE_PUBLIC_KEY` が未設定の場合、決済を「デモ完了」で保存します。
