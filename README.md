# kokokoko

和風でクールかつ高級感のある、タトゥー予約サイト（静的デモ）です。

## コンセプト
- 墨・静けさ・余白を意識した和モダンUI
- 在籍彫師 / アーティストプロフィール登録 / 料金目安 / 予約フォームを1ページで完結
- 予約リクエスト・アーティストプロフィールは `localStorage` に保存して一覧表示
- ロゴ画像（`assets/inklink-logo.svg`）をトップ表示とホーム戻りボタンに使用

## 使い方
1. `index.html` をブラウザで開く
2. 「アーティストプロフィール登録」または「予約フォーム」を入力して送信
3. 右下ロゴボタンでトップへ戻る
